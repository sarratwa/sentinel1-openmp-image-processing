Dieser Ordner enthält Python-Hilfsskripte für die Vorbereitung und Auswertung des Projekts.

Geplante Aufgaben der Skripte:

- Vorbereitung oder Konvertierung von Bilddaten
- Erzeugung synthetischer Testbilder
- Auswertung der Benchmark-Ergebnisse
- Erstellung von Diagrammen für Laufzeit, Speedup und Effizienz

Beispiel:

```bash
python scripts/plot_results.py
