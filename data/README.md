# Data

Dieser Ordner enthält vorbereitete Eingabedaten für die Bildverarbeitung.

Die ursprünglichen Sentinel-1 SAR-Daten werden nicht direkt im C/OpenMP-Programm verarbeitet. Stattdessen werden sie zuerst in ein einfaches Arbeitsformat, z. B. PGM, TIFF oder RAW, konvertiert.

Geplante Inhalte:

- vorbereitete Eingabebilder
- kleine Testbilder für die Entwicklung
- größere Testbilder für Benchmarks

Große Datensätze werden eventuell nicht direkt im Repository gespeichert. In diesem Fall werden hier Download- und Vorbereitungshinweise ergänzt.
